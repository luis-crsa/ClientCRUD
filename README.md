# Tecnologias utilizadas
- Java
- Spring Boot
- JPA / Hibernate
- Maven

# Como executar o projeto
Pré-requisitos: Java 21

```bash
# Clonar repositório
git clone https://github.com/luis-crsa/ClientCRUD.git

# Acessar a pasta do projeto
cd clientcrud

# Executar o projeto
./mvnw spring-boot:run
```
